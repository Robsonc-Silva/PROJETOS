# Repositório de Projetos

Este repositório reúne arquivos e documentos relacionados a projetos desenvolvidos na faculdade.  
A organização segue uma estrutura por ano, facilitando a navegação e consulta:

- **ANO2025**: atividades e trabalhos realizados em 2025  
- **ANO2026**: atividades e trabalhos realizados em 2026  

## Objetivo
Centralizar e compartilhar tarefas e projetos acadêmicos com professores e colegas, mantendo uma organização clara e acessível.

## Estrutura
Cada pasta contém:
- Documentos (.docx, .pdf, etc.)
- Planilhas (.xlsx)
- Outros materiais de apoio

## Observações
- Este repositório é de uso acadêmico.  
- Os arquivos podem ser atualizados conforme novos projetos forem concluídos.
